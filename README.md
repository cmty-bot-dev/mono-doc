> REST API Documentation Generator for [Mono](https://github.com/terrajs/mono)

## Installation

``` bash
npm install -g mono-doc
```

## Usage

Start documentation server on http://localhost:3000 for `<api-url>`:

```bash
mono-doc <api-url>
```
